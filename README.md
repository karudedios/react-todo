# React ToDo

Built with ES7 and Flow.

## Install

|          |               |
|----------|---------------|
| **Yarn** | `yarn add`    |
| **Npm**  | `npm install` |

## Run in Browser

|          |                 |
|----------|-----------------|
| **Yarn** | `yarn serve`    |
| **Npm**  | `npm run serve` |
